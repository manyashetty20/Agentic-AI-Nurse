Project Components Overview

  • Frontend (UI): HTML/CSS/JS (`index.html`). Provides the instant, hard-coded interview logic, Voice/Text input, and final report display.
  • Backend (API): Python FastAPI (`main.py`). Manages communication, holds the hard-coded interview script, and orchestrates the AI calls.
  • AI Core (RAG): LangChain/BioMistral. Uses the `knowledge_base` (RAG) to generate clinically accurate Differential Diagnoses (DDx) and Red Flags.
